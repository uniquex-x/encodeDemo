# Vorbis声道耦合改进方案

## 问题描述
原始的Vorbis声道耦合在高频部分（14k~16kHz）会产生高频爆音（High Frequency Boost），类似铙钹的金属声，特别是在频率动态范围较大的音乐样本中。

## 改进方案

### 1. 高频爆音缓解函数 (`hf_boost_mitigation`)
- **功能**: 对14kHz以上的频率进行渐进式衰减
- **机制**: 在14-16kHz范围内，衰减因子从0.7逐渐降至0.3
- **目的**: 减少高频部分的过度增强

### 2. 增强型无损耦合 (`couple_lossless`)
- **改进**: 保留部分角度信息以减少相位失真
- **限制**: 角度值限制在±0.5范围内，防止过度相位扭曲
- **优势**: 相比完全丢弃角度信息，提供更好的立体声成像

### 3. 改进的预计算点耦合 (`precomputed_couple_point`)
- **基于**: JingZhang的研究成果
- **特点**: 使用查找表优化计算效率
- **目标**: 在有损耦合中提供更精确的幅度计算

### 4. 立体声宽度保护 (`preserve_stereo_width`)
- **频率范围**: 12-16kHz
- **保护机制**: 保留30%的立体声宽度信息
- **目的**: 减少高频立体声成像的损失

### 5. 主耦合逻辑改进
- **自适应处理**: 根据频率自动选择最佳耦合策略
- **高频保护**: 在高频段应用特殊的缓解措施
- **立体声保持**: 在关键频段保持立体声特性

## 技术特点

### 频率感知处理
```c
float freq = (float)((i+j) * p->rate) / (2.0f * n);
```
每个频率箱都根据其实际频率值进行针对性处理。

### 渐进式衰减
```c
float hf_factor = (freq - 14000.0f) / 2000.0f;
return 0.7f - 0.4f * hf_factor;
```
避免突变，提供平滑的频率响应过渡。

### 立体声成像保护
```c
preserve_stereo_width(&reM[j], &reA[j], freq, 1.0f);
```
在高频段保持必要的立体声信息。

## 预期效果

1. **减少高频爆音**: 显著降低14-16kHz范围内的金属声和刺耳感
2. **保持音质**: 在低频和中频段保持原有的编码效率和音质
3. **立体声改善**: 改善高频段的立体声成像效果
4. **兼容性**: 与现有Vorbis标准完全兼容

## 适用场景

- 古典音乐（含铜管乐器和打击乐器）
- 摇滚音乐（含铙钹等金属打击乐器）
- 高动态范围音频内容
- 对高频细节敏感的音频应用

## 配置建议

- **低质量设置(q≤5.99)**: 启用混合耦合，重点使用高频缓解
- **高质量设置(q≥6)**: 主要使用增强无损耦合，保持立体声宽度
- **实时应用**: 可根据CPU性能调整缓解强度

这些改进在保持Vorbis编码效率的同时，显著改善了高频音质，特别适合处理包含丰富高频内容的音乐。
